Welcome to BioDevOps Pages.

BioDevOps project will provide DevOps tools for bioinformatics researchers. Bioinformatics researchers should focus to develop a novel bioinformatics algorithm or analysis biological data. However, we must consume most of ours time on setting up a computer and data analysis environments.

Our solutions

We will provided ...

  • chef cookbooks (berkshelf) and testing configuration (test-kitchen) for installing various bioinformatics tools
  • virtual machine and Amazon Machine Image for bioinformatics
  • provisioning framework for virtual PC cluster into cloud computer or a virtualization system by using vargent
